大家知道为什么小学要开始学英语么？因为当年这些老师和家长在中学阶段被英语整得欲仙欲死，一直怨恨自己基础不好，于是，突发奇想，觉得如果小学就有基础，中学英语就不会差了……

同样逻辑，很多以前中学或者大学才需要学的事情，都提前到早教了……

其实，在合适年龄段学习，没学好，主要是耐性和刻苦，而不是因为基础不好。
